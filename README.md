# Benchmaring Quantum Devices

## Installation

```shell
git clone https://github.com/nathanieltornow/qpu_bench.git && cd qpu_bench
```
```shell
python3 -m venv .venv
```
```shell
source .venv/bin/activate
```
```shell
pip install --upgrade pip
```
```shell
pip install .
```

Find the examples in the [examples](examples) folder.
